# AI Proposal Writer

A single-page web application for AI-driven proposal writing built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS

## Usage

Fill in the project details form and click "Generate AI Proposal" to create your proposal. You can then copy or download the generated proposal.
